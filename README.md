# Amazon Book Analysis

This project analyzes Amazon bestselling books from 2009 to 2019. It explores trends in book genres, authors, reviews, and popularity.

## Dataset

'bestsellers_with_categories.csv' — contains book names, authors, ratings, prices, reviews, genres, and publication year.

## Features

1. Data cleaning and preparation
2. Added features:
3. 'name_len': number of characters in book names (excluding spaces)
4. 'punc%': percentage of punctuation in book names
5. Categorized prices into ranges

## Analysis & Visualizations

1. Distribution of books per genre over the years
2.Top 15 authors by reviews
3. Popular books and their genres
4. Price range vs book popularity
